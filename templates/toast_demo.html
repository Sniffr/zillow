Toast Examples
Toast Features:
  • Auto-dismiss after specified duration
  • Manual close button
  • Different types (success, error, warning, info)
  • Responsive design for mobile
  • Beautiful animations
  • Icons for each message type
Usage Examples
Basic Usage:
// Show different types of toasts
showSuccessToast('Operation completed successfully!');
showErrorToast('Something went wrong!');
showWarningToast('Please check your input!');
showInfoToast('Here is some information!');

// Custom toast with duration
showToast('Custom message', 'info', 8000);
In Your Code:
// Replace JavaScript alerts
// Instead of: alert('Success!');
showSuccessToast('Success!');

// Instead of: confirm('Are you sure?')
if (confirm('Are you sure?')) {
    // Do something
    showSuccessToast('Action completed!');
}
